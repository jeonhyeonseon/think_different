# 🍎 Think_Different  🍎

## 소개
Spring Boot 기반 개인 웹 서비스입니다.
회원 인증을 기반으로 게시판 / 실시간 채팅 / 가계부 기능을 하나의 서비스로 통합해 구현했습니다.

## 핵심 기능
### 인증
회원가입 / 로그인 / 로그아웃

### 게시판
게시글 CRUD

검색 / 페이징

### 실시간 채팅 (WebSocket + STOMP)
채팅방 구독/발행

메시지 전송/수신

### 가계부
지출/수입 CRUD

카테고리 분류

월별 조회

## 기술 스택
Backend: Java, Spring Boot, Spring MVC, JPA, Spring Security

Front: Thymeleaf, HTML/CSS/JS

Realtime: WebSocket, STOMP

DB: MySQL
